题目：

1748 唯一元素的和：

给你一个整数数组 nums 。数组中唯一元素是那些只出现 恰好一次 的元素。

请你返回 nums 中唯一元素的 和 。

题目原链接：https://leetcode-cn.com/problems/sum-of-unique-elements/

2006 差的绝对值为 K 的数对数目：

给你一个整数数组 nums 和一个整数 k ，请你返回数对 (i, j) 的数目，满足 i < j 且 |nums[i] - nums[j]| == k 。

​ |x| 的值定义为：

​ 如果 x >= 0 ，那么值为 x 。
​ 如果 x < 0 ，那么值为 -x 。

题目原链：https://leetcode-cn.com/problems/count-number-of-pairs-with-absolute-difference-k

1748：

思路：一次遍历、bool数组标志

  1. 定义两个bool数组，用nums中的各个元素值num作为下标 (1<= num <= 100)；

  2. 遍历过程中遇到frequencyFalg不为真的num,判断其minusFlag是否为真：

    • 若是则略过（之前已在结果res中相减）；

    • 不是则res减去该num，并改变对应的minusFlag；

  3. 若frequecyFlag为假，则将之相加，改变对应frequencyFlag；

2006：

思路：map简单应用

  1. 判断|nums[i]-nums[j]|=k是否成立，即判断nums[i]+k==nums[j] / nums[i]-k ==nums[j]这两个式子中是否有一个成立；

  2. 因为i<j，对数组里较后的元素j来说，我们仅需判断j之前的若干个元素i里有几个满足上述两个式子（nums[i]+k==nums[j]……）；

  3. 为了实现上述目的，我们定义一个map，nums[i]+k,nums[i]-k为键，出现次数为值。对nums遍历，其中的每一个元素num，均将num+k和num-k存入map中，同时判断map是否存在键num，存在则意味着在num之前的元素里有满足两个式子的，我们把其出现次数加入结果。最终遍历完整个数组，我们能得到结果。
