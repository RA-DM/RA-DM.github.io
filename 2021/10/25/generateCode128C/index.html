ruby实现code128C条形码

生活中我们时常能看到条形码，但之前我却一直没想过它的实现方式，正巧课程有个ruby项目要做，就写了个ruby版本的code128C条形码生成。在本文及代码写作过程中，下面这位作者的博客给了我很大启发，个人觉得他的这篇是这方面讲的相对透彻的。本文如有不理解的地方，可以移步至这看大佬的详细讲解

code128C原理简要说明：

CODE128码是广泛应用在企业内部管理、生产流程、物流控制系统方面的条码码制，由于其优良的特性在管理信息系统的设计中被广泛使用。其可以对全部128个ASCII字符（包括数字、大小写字母、符号和控制符）进行编码。其有三种不同的版本：A、B、C。

  • CODE128A:标准数字和大写字母,控制符,特殊字符

  • CODE128B:标准数字和大写字母,小写字母,特殊字符

  • CODE128C:[00]-[99]的数字对集合,共100个

​ -引用自百度百科和维基百科

假设先定义条形码的基本组成单位为一条长宽适中的矩形块(如下图中的unit)，颜色为黑或白; 则整个条形码由若干个这个基本单位组成。

code128编码将每个ASCII字符和控制位（如开始位）映射成数量若干的基本单元（各自单元颜色也已定义好，如下图的startC开始位）；对要给定的字符串，其每个字符映射的条块和相应的校检位、开始结束条块拼接在一起就组成了其对应的code128条形码。

require "ruby2d"

$codeC_map=Hash.new
$codeA_to_codeC=Hash.new

#codeA hash to code C's index, only for 0-9
for i in 0..9
    $codeA_to_codeC[i]=i+16
end

#input the codeC mapping dict
file = File.open("code128C_mapping.txt") 
#包含了code128C映射的txt文件，即上文中表编码值那列，可自己创建

i=0
file.each_line do |line|
    $codeC_map[i]=line
    i+=1
end
puts $codeC_map[101]

file.close

#generate code
def generateCode128c(s)
    if s==""
        puts "Error: input is null"
        return ""
    end
    result=$codeC_map[105]
    checkBit=105
    numbers = []

    #even length without code128A
    if s.length%2==0
        i=1
        s.chars.each_slice(2) do |slice|
            index=Integer(slice.join)
            checkBit+=index*i
            result+=$codeC_map[index]
            i+=1
        end
        result+=$codeC_map[checkBit%103]+$codeC_map[106]
        return result
    
    #odd length with code128A
    else
        i=1
        s[0..-2].chars.each_slice(2) do |slice|
            index=Integer(slice.join)
            checkBit+=index*i
            result+=$codeC_map[index]
            i+=1
        end
        result+=$codeC_map[101]+$codeC_map[$codeA_to_codeC[Integer(s[-1])]]
        checkBit+=101*i+$codeA_to_codeC[Integer(s[-1])]*(i+1)
        result+=$codeC_map[checkBit%103]+$codeC_map[106]
        return result
    end
end

#generate picture
def generatePicture(s)
    set background: 'white'
    code=generateCode128c(s)

    if code.length==0
        puts "Error: the generateCode function failed."
    end

    for i in 0..code.length-1
        if code[i]=='b'
            Rectangle.new(x:200+i*2,y:150,width:2,height:135,color:'black')
        elsif code[i]=='s'
            Rectangle.new(x:200+i*2,y:150,width:2,height:135,color:'white')
        end
    end

    Text.new(
        s,
        x: 150+(code.length-1), y: 300,
        size: 15,
        color: 'black',
    )
    show
end
